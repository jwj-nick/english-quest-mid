**Live:** https://jwj-nick.github.io/english-quest-mid/

---

## 처음 시작하기 (한 번만)

1. 위 URL을 폰 크롬/사파리에서 연다.
2. 메뉴 → **"홈 화면에 추가"** 누른다. (PWA — 앱처럼 동작)
3. 첫 실행 시 **프로필**을 만든다:
   - 이름 입력
   - 아바타 12종 중 선택
   - 직업 4종 (마법사 / 전사 / 궁수 / 학자) 중 선택
   - 포인트 색 5종 중 선택
4. 끝. 다음부터는 그냥 홈 화면 아이콘만 누르면 된다.

## 일일 미션 (홈 카드)

매일 자동 갱신. 3개를 다 완료하면 그날 종료.

1. 🌳 **어휘의 숲 1회** — (+10 XP 보너스 표시)
2. ✨ **아무 던전 1회** — (+5 XP 보너스)
3. 🎯 **100% 정답 1회** — (+15 XP 보너스)

> ※ 보너스 XP는 현재 **표시만**이고 실제 추가 지급은 안 돼. 회당 XP는 정답률로 이미 계산되어 들어가.

## 어휘의 숲 — 단어 풀 선택

어휘 던전 진입 시 **3개 pill 버튼**이 보여:

- **전체** — 이 주차 모든 단어 (10개)
- **오늘 복습** — 며칠 지나서 다시 봐야 할 단어 (간격 반복, SR)
- **약한 단어** — 자주 틀린 단어 우선

처음에는 "전체"만 활성. 며칠 사용한 뒤 "오늘 복습"/"약한 단어"가 활성화돼.

## ✅ 매일 체크리스트

```
□ 앱 열기
□ 어휘의 숲 1회 (3~5분)
□ 다른 던전 1회 (3~5분) — 듣기/독해/말하기/쓰기 골고루
□ 일일 미션 3개 완료 확인
□ 7일 연속 → 🔥 일주일 모험가 배지
```

소요시간 **5~10분 목표**. 부담 없이.

---

## ✅ 주말 체크리스트

```
□ 한 주 누적 XP 확인 (📊 기록 페이지)
□ 약점 단어 풀이 비어있는지 확인 (있으면 한 번 더 돌리기)
□ 보스전 (있는 주차) — 주말 미니게임
□ (선택) 학습 로그 JSON 내보내기 → 아빠에게 전달 (다음 주 콘텐츠 보강용)
```

## 자주 묻는 것

**Q. 기록은 어디 남아?**
A. 폰 브라우저 내부 (IndexedDB). 인터넷 안 되어도 기록돼. 다만 브라우저 데이터를 지우면 사라지니까 가끔 프로필 → "학습 로그 JSON 내보내기" 하면 백업 가능.

**Q. 다른 폰/PC 에서 보면?**
A. 각 기기마다 따로 기록돼 (sync 없음, V1). 한 기기만 정해놓고 쓰는 게 좋아.

**Q. 앱이 갱신 안 돼.**
A. 폰 브라우저에서 한 번 새로고침 (당기기) 하거나, PWA 라면 앱 종료 후 재실행. 보통 다음 진입 시 자동 업데이트.

**Q. 콘텐츠가 다 떨어졌어.**
A. 한 주차에 어휘 10개 / 듣기 2 / 독해 2 / 말하기 3 / 쓰기 1 / 보스전 1 — 충분히 반복해도 되고, 다음 주차가 나오면 자동 추가돼.

**Q. 단어 발음 안 들려.**
A. 브라우저의 **Web Speech API** 사용 — 폰의 시스템 영어 TTS 가 있어야 동작. iOS는 보통 OK, 안드로이드는 "Google TTS" 설치 필요할 수 있어.
